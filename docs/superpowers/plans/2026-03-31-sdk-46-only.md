        val sdkDir = project.layout.buildDirectory.dir("sdk").get().asFile

- [ ] **Step 2: prepare-Task auf SdkBuilder umstellen**

In `gdscript/build.gradle.kts`, den `register("prepare")` Block (Zeile 80-109) ersetzen:

```kotlin
register("prepare") {
    doLast {
        val sdkDir = project.layout.buildDirectory.dir("sdk").get().asFile
        val sdkFile = sdkDir.resolve("sdk.tar.xz")
        if (sdkFile.exists()) return@doLast

        SdkBuilder.build(sdkDir)
        logger.lifecycle("SDK built to ${sdkFile.absolutePath}")
    }
}
```

Hinweis: Der genaue Aufruf haengt von der API ab, die der Kotlin SDK-Builder exponiert. Ggf. anpassen.
